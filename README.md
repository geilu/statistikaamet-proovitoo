# Statistikaameti proovitöö

Proovitöö ülesandeks oli teha interaktiivne viktoriinirakendus.

---

## Käivitamine

#### 1. Kloonida repo ja liikuda selle kausta
```bash
git clone https://github.com/geilu/statistikaamet-proovitoo
cd statistikaamet-proovitoo
```
#### 2. Käivitada rakendus
```bash
npm install
npm run dev
```
#### 3. Ava rakendus
Browseris mine lingile http://localhost:5173.
